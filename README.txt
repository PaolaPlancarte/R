# Mini Dashboard R Project (macOS / Posit Cloud / Jupyter compatible)

## Files
- sales_demo.csv
- dashboard_R.R
- README.txt

## Quick Start (Posit Cloud)
1) Create a new project.
2) Upload the three files.
3) Install required libraries if needed:
   install.packages(c("ggplot2", "dplyr", "readr"))
4) Run:
   source("dashboard_R.R")

## Output:
- KPI: displayed in console
- Chart: donut_by_category.png (downloadable)

You can replace sales_demo.csv with real data (matching the same column names).
